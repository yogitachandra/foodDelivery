## foodDelivery

Create a automated test for the search feature
Selenium webdriver,Java and TestNG is used to solve this challenge. 
PageObjectModel, a design pattern is used

## Resources

Prerequisite

Java 1.8 - http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

IDE used Eclipse neon - http://www.eclipse.org/downloads/

Selenium 3.5.3 http://seleniumhq.github.io/selenium/docs/api/java/index.html

TestNG - Go to eclipse and download the plugin from eclipse marketplace

## Solution compatibility

Windows 

To change the driver or app url - test.xml

## Instructions to run

1. Install java 1.8 - Java paths should be set . To verify if it works fine , Open command prompt and type java.
1. Install Eclipse 
1. Install TestNG 
1. Clone the repository in your local machine 
1. Import the project in eclipse
1. Right click and configure build path for selenium jar
1. Right click on testRun.xml and do Run as > TestNG Suite
1. Report location - \FoodProject\test-output\Page Object test example\sample test.html

## Scope
1. Default country is Germany and Language is English (can be enhanced later)
1. Supported browser - Google chrome
1. Can only be tested on web browser
1. Feature tested is search field only on homepage
1. Out of scope - Non functional testing like Performance, Load, Stress
